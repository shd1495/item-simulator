# 아이템 시뮬레이터
## 기술 스택
- Node.js
- Express.js
- MySQL - Prisma ORM
- JWT
- AWS EC2
## 실행 방법
- 서버 - yarn run start
- API - shd1495.store/api/url
## 필수 기능
 - [x] 회원가입 API

 - [x] 캐릭터 삭제 API (JWT 인증만 가능)
 - [x] 캐릭터 상세 조회 API (JWT 부분 인증)
 - [x] 아이템 생성 API

 - [x] 아이템 목록 조회 API
 - [x] 아이템 상세 조회 API

 - [x] 아이템 구입 API (JWT 인증만 가능)
